# [CN200](https://www.nitrc.org/projects/us200_cn200)

## 构建方法

* 样本包含19-37岁200名(男100)被试的T1w MRI图像
* 包含所有样本构建的头/全脑和组织概率模板，以及男女样本分开的模板
* 模板初始化空间位于MNI152空间
* 预处理采用HCP minimal preprocessing pipelines；使用FSL FAST进行组织分割；模板构建使用ANTs
