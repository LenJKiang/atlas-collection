# [Harvard-Oxford](https://fsl.fmrib.ox.ac.uk/fsl/fslwiki/Atlases)

## 构建方法

* 根据37名被试半手动分割得到的概率地图，包含48个皮层脑区（不区分左右）和19个皮层下区域（包含白质和侧脑室）。
